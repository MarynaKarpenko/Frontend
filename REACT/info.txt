1. Установка react выполнить только один раз
npm i -g create-react-app

2. Установка приложения (выполнять каждый раз) (Конфигуратор)
npx create-react-app myapp

3. Запуск
a)cd myapp (ОБЯЗАТЕЛЬНО)
b)npm start

Чтобы выключить приложение, необходимо написать CTRL+C

(в приложении можно в любой момент удалить папку node_modules)
ее можно востановить 
npm i
